# Dokumenty / oferty PDF

Wrzuć tutaj swój gotowy, zaprojektowany plik oferty pod nazwą:

```
oferta.pdf
```

Ten plik jest automatycznie dołączany jako załącznik do każdego e-maila
wysyłanego po wypełnieniu formularza lub konfiguratora.

Aby użyć innej nazwy/ścieżki, ustaw zmienną `OFFER_PDF_PATH` w `.env.local`
(ścieżka względem katalogu głównego projektu), np.:

```
OFFER_PDF_PATH=public/documents/moja-oferta.pdf
```
